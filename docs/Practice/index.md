# 练习题目

这些练习题目来自于公开的做题网站，提供给选课同学做额外练习使用，顺便也提供给旁听的同学练习使用。

建议用C/C++语言完成这些练习题。

这些题目在网上有大量公开的解答分享，可以有两种用法：

1. 独立尝试完成这些题目。缺乏思路、调试困难时上网搜索公开的解答，用于提示思路，或用于对拍。
2. 直接搜索公开的解答进行代码阅读，学习其中的思路，然后再尝试自己把程序写一遍。
